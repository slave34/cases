# cases
Мы с моим напарником создадим сайт симулятор открытия кейсов. Главной страничкой займеться напарник, на ней будет выбор из 4 различных кейсов для открытия и соотвественно кнопки для перехода на само открытие. 
Второй страничкой займусь я, на которой будет происходить само открытие, то есть анимация. Также будет кнопка для перехода обратно на главную страницу.
